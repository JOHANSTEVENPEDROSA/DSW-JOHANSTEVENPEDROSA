Listas y Enlaces


Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.



Flex y Box


El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación.



En este Sitio encontraras informacion para poder desarrollar una pagina web en Html aplicando las Listas y enlaces , tambien encontraras informacion acerca de los flex y box muy comunmente usados en el diseño de sitios web, ademas encontraras contenido multimedia para tu mañor entendimiento.
